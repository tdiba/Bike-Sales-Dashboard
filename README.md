### Project Overview
This is an interactive Bike Sales Dashboard built in Microsoft Excel to analyze customer demographics and factors influencing bike purchases. The dashboard provides insights into how income level, age group, gender, and commute distance affect purchasing behavior. It enables quick filtering and exploration of the dataset to identify patterns among different customer segments.

### Objectives
The goal of this project was to:
- Explore bike purchase behavior using customer demographic data
- Build an interactive Excel dashboard
- Identify patterns that influence bike purchases
- Present insights in a clear visual format



### Tools and Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design


### Dataset
The dataset contains demographic and lifestyle information about customers, including:
- Gender
- Marital Status
- Income
- Education Level
- Region
- Number of Children
- Age Bracket
- Commute Distance
- Bike Purchase Status (Yes/No)

These variables were used to analyze purchasing trends.



### Dashboard Features
1. Average Income of Bike Purchasers
   This chart compares the average income by gender for customers who purchased a bike versus those who did not.
   Insights from this chart include:
   - Customers who purchased bikes tend to have higher average incomes.
   - Male purchasers have slightly higher income levels than female purchasers.



2. Customer Age Brackets
   This visualization shows how bike purchases vary across different age groups.
   Here it can be seen that Middle-aged customers represent the largest group of bike purchasers.


3. Customer Commute Distance
   This chart analyzes how commuting distance influences bike purchases.
   Here it can be seen that customers with shorter commute distances are more likely to purchase bikes.


4. Interactive Filters
   The dashboard includes slicers that allow users to filter the data.
   These filters make the dashboard fully interactive and allow users to explore different customer segments.


### Data Analysis Process
1. Data Cleaning
   - Removed Duplicates
   - Removed unnecessary columns
  
2. Data Transformation
   - Created calculated fields
   - Formatted Marital Status and Gender Columns to show full status
   - Organized data for pivot analysis
  
3. Data Aggregation
   - Used Pivot Tables to summarize data
  
4. Dashboard Creation
   - Built visualizations using Pivot Charts
   - Added slicers for interactivity
   - Designed a clean dashboard layout


### Key Insights
  - Bike purchasers generally have higher average incomes.
  - Middle-aged customers are the most likely to purchase bikes.
  - Short commute distances correlate strongly with bike purchases.
  - Customer demographics such as education and region can influence purchasing behavior.
